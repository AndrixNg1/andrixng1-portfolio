# 🚀 Andrix Ng Portfolio

A high-performance, dark-themed portfolio built with **Astro** and **native CSS**. This project serves as the digital resume and project showcase for Andrix Ng, a Full-Stack Developer specializing in modern web architectures and premium user interfaces.

The design is heavily inspired by professional and minimalist aesthetics, ensuring a seamless user experience with fluid animations and responsive layouts.

## ✨ Features

- **Modern Stack**: Built with Astro for extreme performance and fast load times.
- **Data-Driven Content**: Profile information, projects, and page content are strictly typed and managed via TypeScript data files in `src/data`.
- **Custom Styling**: Leveraging native CSS for complete control over design tokens, animations, and typography without the overhead of heavy frameworks.
- **SEO Optimized**: Fully static, fast, and SEO-friendly structure out of the box.

## 🛠️ Technology Stack

- **Framework**: [Astro](https://astro.build)
- **Styling**: Native CSS
- **Typing**: TypeScript

## 🚀 Quick Start

To run the project locally, follow these steps:

### Prerequisites
Make sure you have Node.js (>=22.12.0) installed.

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

The local development server will start at `http://localhost:4321`.

### Build for Production

```bash
# Build the static site for deployment
npm run build
```

## 📁 Project Structure

```text
src/
├── components/   # Reusable UI components (buttons, cards, etc.)
│   └── layout/   # Layout-specific components (navbar, footer, etc.)
├── data/         # TypeScript files containing content (profile, projects, pages)
├── layouts/      # Main Astro layout wrappers
├── pages/        # Astro routing pages
└── styles/       # Global CSS stylesheets
```
