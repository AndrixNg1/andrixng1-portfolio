# Andrix Ng Portfolio

Portfolio Astro sombre inspire du design `OpenDayHPC`, adapte pour presenter le profil, les projets, les competences et le parcours de Andrix Ng.

## Stack

- Astro
- CSS natif
- Donnees TypeScript dans `src/data`

## Commandes

```bash
npm install
npm run dev
npm run build
```

Le serveur local utilise par defaut `http://localhost:4321`.

## Structure

```text
src/
├── components/layout/
├── data/
├── layouts/
├── pages/
└── styles/
```
